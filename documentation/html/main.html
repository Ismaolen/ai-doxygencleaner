main.main module¶

Main execution script for the Doxygen Warning Manager tool.

The AI Doxygen Cleaner is a tool that automates several tasks related to Doxygen warning management. This includes configuring Doxygen, detecting warnings, inserting warnings into a database, fixing warnings using an AI language model, applying these fixes, detecting warnings after fixes have been applied, printing warning details to a file, generating a diff page in HTML format, and updating warning data in the database post-fix.
